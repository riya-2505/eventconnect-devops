# EventConnect Pro 🚀

A DevOps-based event management platform.

## Features
- Create and manage events
- Real-time event listing
- Docker support
- CI/CD ready

## Tech Stack
- Node.js
- HTML, CSS, JS
- Docker

## Run Locally
npm install
npm start
